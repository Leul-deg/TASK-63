

# Business Logic Questions Log

This document records prompt ambiguities and the implementation choices made in `repo/`.
It is intended to satisfy the requirement that important assumptions be written down rather than hidden in code.

---

### 1. Who is the primary actor for the portal?

**Question:** The prompt lists multiple possible actors (front-desk operator, manager, admin) but does not define the primary user.
**My Understanding:** The system is operational in nature and will be used by multiple roles depending on context.
**Solution:**

* The portal supports multiple primary actors:

  * front-desk operators (day-to-day usage)
  * managers (oversight and reporting)
  * administrators (configuration and control)
* No single “primary” actor is enforced; RBAC governs capabilities.
